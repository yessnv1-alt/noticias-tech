# 🌐 Tech & Science News Explorer

Una aplicación web moderna y minimalista que permite consultar las últimas noticias del mundo de la tecnología y la ciencia. El proyecto utiliza la API de NewsData para obtener información actualizada y permite al usuario descargar un resumen profesional en formato de texto.

## 🚀 Características
- **Consumo de API Real:** Uso de `fetch` para obtener noticias en vivo.
- **Interfaz Moderna:** Diseño oscuro (Cyberpunk vibes) con bordes amarillos, animaciones hover y diseño responsive gracias a **Bootstrap 5**.
- **Seguridad:** Manejo de variables de entorno con **Vite** para proteger las credenciales de la API.
- **Exportación de Datos:** Función personalizada en JS para generar un archivo `.txt` con el Top 10 de noticias filtradas.
- **Interactividad:** Iconos dinámicos de Font Awesome y estados interactivos en las tarjetas.

## 🛠️ Tecnologías utilizadas
* **Frontend:** HTML5, CSS3 (Custom Styles), JavaScript Puro (ES6+).
* **Framework CSS:** Bootstrap 5.
* **Herramienta de Construcción (Bundler):** Vite.
* **API:** [NewsData.io](https://newsdata.io/).
